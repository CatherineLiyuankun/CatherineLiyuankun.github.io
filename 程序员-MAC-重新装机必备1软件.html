程序员-MAC 重新装机必备1软件

Posted by Yuankun Li on 2021-05-31
岚

    • Home
    • Archives
    • Categories
    • About
    • Tags

    编程必备

    • [x] Google Chrome 网址下载，或者[APP Store + 国外账号] + Google Chrome Canary + Firefox + Safari，浏览器，调试用，IE 的测试会借助内网的云测平台

    IDE

    • Xcode [APP Store]
    • Vscode + 插件Settings Sync （用gitHub账号创建gist同步配置和插件）。
      • Markdown Extension Pack
    • Toolbox + IntelliJ

    编程辅助

    • SourceTree git 辅助，由于 git 高级操作命令记不住，就只用借助 UI 了。
    • Dash 快速查API文档。免费版本会在你搜索完后等10秒再显示，付费版：$29.99 无需等待。
    • ColorSnapper2，取色工具。付费，App store 没有。
      https://cyberduck.io/
    • Cyberduck

    网络

    • Paw，请求模拟，前后端联调API时用这个先走一遍。
    • Postman，请求模拟，前后端联调API时用这个先走一遍。
    • Charles，抓包用，支持 https, 不免费，公司提供的付费账号。
    • Gas Mask ，Hosts 管理

    命令行工具

    • Iterm2

    效率

    • [x] Alfred + Powerpack 【我的百度网盘、硬盘】应用启动、粘贴板管理、Workflow、Snippets 管理等。 同步问题可以参考：Alfred 中如何在不同电脑上同步自定义设置？ 和 在Mac之间同步您的Alfred设置。免费功能：File search, Web search，app启动, Calculator, Dictionary, System。付费功能（需购买Powerpack）：Workflow，Action, Clipboard History，Snippets, Contacts, itunes, 1Password, Terminal. 【我的百度网盘、硬盘】

    • [x] Thor，【网址下载】或者[APP Store + 国外账号]一键直达应用， 免费，开源。

    • Karabiner Element，用于把右 Command 和 Capslock 键利用起来，避免快捷键冲突，简单 note，详见sorrycc的《我的快捷键技巧》. 我试了后, 把“右 Command”改成F19,发现我右 Command+delete还是经常用的，所以接着在Karabiner Element 的“Simple modifications” 把 “right_option” 改成了 “left_command”。遇到了macOS 10.14.6 Karabiner-Elements 疑似无法正常使用的问题. 另外参考Mac 自定义应用程序快捷键配置了Mac app，比如Chrome和iterm3的一些快捷键。免费，开源。

    • [x] KeepingYouAwake，可保证Mac系统不自动休眠。原来用的Caffeine，但是MacOS更新到Mojave以后就不管用了。可以改用Amphetamine 或者 KeepingYouAwake。免费，开源。

    • TickTick滴答 ，任务管理，有MAC版和iPhone版，可以同步。[APP Store]

    • [x] CleanMyMac 3 【我的百度网盘、硬盘】系统清理。

    • [x] RescueTime和WakaTime来记录你的时间，都有Chrome插件。RescueTime 手机端电脑端都有app。

    工具

    • 百度网盘
    • office 365
    • [x] Kiwi for Gmail，Gmail 客户端, 我用的免费的lite版，App store 要切换到美国。
    • [x] Microsoft NTFS for Mac by Paragon Software - 支持 NTFS 格式硬盘，因为原来买了希捷的移动硬盘，送的软件，一般、够用。

    输出

    • [x] Onenote，笔记工具，从 Evernote印象笔记 （原来是会员，现在只是免费版） 切到 Onenote，感觉Onenote对markdown支持不好（只有windows版本有插件可以支持markdown），准备换Ulysses。不过我现在markdown一般都用vscode+github来写了。

    • [x] OmniGraffle + Xmind

    • [x] Gif Brewery 3 ，GIF 录屏工具, 可以直接录屏并转换成gif动图。免费，App store 下载。

    输入

    • [x] Reeder，RSS 阅读软件

    虚拟机

    • [x] ParallelsDesktop

    工作必备

    • [x] GlobalProtect 【网址下载】 工作 VPN 软件
    • [ ] Microsoft Remote Desktop [APP Store]
    • [ ]

    生活娱乐必备

    • 各类视频软件
