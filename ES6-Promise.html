    Promise

    Promiseжһ„йҖ еҮҪж•°жҺҘеҸ—дёҖдёӘеҮҪж•°дҪңдёәеҸӮж•°пјҢиҜҘеҮҪж•°зҡ„дёӨдёӘеҸӮж•°еҲҶеҲ«жҳҜresolveе’ҢrejectгҖӮе®ғд»¬жҳҜдёӨдёӘеҮҪж•°пјҢз”ұ JavaScript еј•ж“ҺжҸҗдҫӣпјҢдёҚз”ЁиҮӘе·ұйғЁзҪІгҖӮ

    resolveеҮҪж•°зҡ„дҪңз”ЁжҳҜпјҢе°ҶPromiseеҜ№иұЎзҡ„зҠ¶жҖҒд»ҺвҖңжңӘе®ҢжҲҗвҖқеҸҳдёәвҖңжҲҗеҠҹвҖқпјҲеҚід»Һ pending еҸҳдёә resolvedпјүпјҢеңЁејӮжӯҘж“ҚдҪңжҲҗеҠҹж—¶и°ғз”ЁпјҢе№¶е°ҶејӮжӯҘж“ҚдҪңзҡ„з»“жһңпјҢдҪңдёәеҸӮж•°дј йҖ’еҮәеҺ»пјӣrejectеҮҪж•°зҡ„дҪңз”ЁжҳҜпјҢе°ҶPromiseеҜ№иұЎзҡ„зҠ¶жҖҒд»ҺвҖңжңӘе®ҢжҲҗвҖқеҸҳдёәвҖңеӨұиҙҘвҖқпјҲеҚід»Һ pending еҸҳдёә rejectedпјүпјҢеңЁејӮжӯҘж“ҚдҪңеӨұиҙҘж—¶и°ғз”ЁпјҢе№¶е°ҶејӮжӯҘж“ҚдҪңжҠҘеҮәзҡ„й”ҷиҜҜпјҢдҪңдёәеҸӮж•°дј йҖ’еҮәеҺ»гҖӮ

    Promiseе®һдҫӢз”ҹжҲҗд»ҘеҗҺпјҢеҸҜд»Ҙз”Ёthenж–№жі•еҲҶеҲ«жҢҮе®ҡresolvedзҠ¶жҖҒе’ҢrejectedзҠ¶жҖҒзҡ„еӣһи°ғеҮҪж•°гҖӮ

    thenж–№жі•еҸҜд»ҘжҺҘеҸ—дёӨдёӘеӣһи°ғеҮҪж•°дҪңдёәеҸӮж•°гҖӮз¬¬дёҖдёӘеӣһи°ғеҮҪж•°жҳҜPromiseеҜ№иұЎзҡ„зҠ¶жҖҒеҸҳдёәresolvedж—¶и°ғз”ЁпјҢз¬¬дәҢдёӘеӣһи°ғеҮҪж•°жҳҜPromiseеҜ№иұЎзҡ„зҠ¶жҖҒеҸҳдёәrejectedж—¶и°ғз”ЁгҖӮе…¶дёӯпјҢз¬¬дәҢдёӘеҮҪж•°жҳҜеҸҜйҖүзҡ„пјҢдёҚдёҖе®ҡиҰҒжҸҗдҫӣгҖӮиҝҷдёӨдёӘеҮҪж•°йғҪжҺҘеҸ—PromiseеҜ№иұЎдј еҮәзҡ„еҖјдҪңдёәеҸӮж•°гҖӮ

    Promise.prototype.then()

    thenж–№жі•зҡ„з¬¬дёҖдёӘеҸӮж•°жҳҜresolvedзҠ¶жҖҒзҡ„еӣһи°ғеҮҪж•°пјҢз¬¬дәҢдёӘеҸӮж•°пјҲеҸҜйҖүпјүжҳҜrejectedзҠ¶жҖҒзҡ„еӣһи°ғеҮҪж•°

    Promise.prototype.catch()

    .then(null, rejection)жҲ–.then(undefined, rejection)зҡ„еҲ«еҗҚпјҢз”ЁдәҺжҢҮе®ҡеҸ‘з”ҹй”ҷиҜҜж—¶зҡ„еӣһи°ғеҮҪж•°гҖӮ

    Promise.prototype.finally()

    finallyж–№жі•з”ЁдәҺжҢҮе®ҡдёҚз®Ў Promise еҜ№иұЎжңҖеҗҺзҠ¶жҖҒеҰӮдҪ•пјҢйғҪдјҡжү§иЎҢзҡ„ж“ҚдҪңгҖӮиҜҘж–№жі•жҳҜ ES2018 еј•е…Ҙж ҮеҮҶзҡ„гҖӮ

    еӨ„зҗҶеӨҡдёӘpromise

    Promise.all()

    return all resolved

    pзҡ„зҠ¶жҖҒз”ұp1гҖҒp2гҖҒp3еҶіе®ҡпјҢеҲҶжҲҗдёӨз§Қжғ…еҶөгҖӮ
    пјҲ1пјүеҸӘжңүp1гҖҒp2гҖҒp3зҡ„зҠ¶жҖҒйғҪеҸҳжҲҗfulfilledпјҢpзҡ„зҠ¶жҖҒжүҚдјҡеҸҳжҲҗfulfilledпјҢжӯӨж—¶p1гҖҒp2гҖҒp3зҡ„иҝ”еӣһеҖјз»„жҲҗдёҖдёӘж•°з»„пјҢдј йҖ’з»ҷpзҡ„еӣһи°ғеҮҪж•°гҖӮ

    пјҲ2пјүеҸӘиҰҒp1гҖҒp2гҖҒp3д№ӢдёӯжңүдёҖдёӘиў«rejectedпјҢpзҡ„зҠ¶жҖҒе°ұеҸҳжҲҗrejectedпјҢжӯӨж—¶з¬¬дёҖдёӘиў«rejectзҡ„е®һдҫӢзҡ„иҝ”еӣһеҖјпјҢдјҡдј йҖ’з»ҷpзҡ„еӣһи°ғеҮҪж•°гҖӮ

    Promise.race()

    return first result

    еҸӘиҰҒp1гҖҒp2гҖҒp3д№ӢдёӯжңүдёҖдёӘе®һдҫӢзҺҮе…Ҳж”№еҸҳзҠ¶жҖҒпјҢpзҡ„зҠ¶жҖҒе°ұи·ҹзқҖж”№еҸҳгҖӮйӮЈдёӘзҺҮе…Ҳж”№еҸҳзҡ„ Promise е®һдҫӢзҡ„иҝ”еӣһеҖјпјҢе°ұдј йҖ’з»ҷpзҡ„еӣһи°ғеҮҪж•°гҖӮ

    Promise.allSettled()

    return all end

    еҸӘжңүзӯүеҲ°жүҖжңүиҝҷдәӣеҸӮж•°е®һдҫӢйғҪиҝ”еӣһз»“жһңпјҢдёҚз®ЎжҳҜfulfilledиҝҳжҳҜrejectedпјҢеҢ…иЈ…е®һдҫӢжүҚдјҡз»“жқҹгҖӮиҜҘж–№жі•з”ұ ES2020 еј•е…ҘгҖӮ
    жҲ‘д»¬дёҚе…іеҝғејӮжӯҘж“ҚдҪңзҡ„з»“жһңпјҢеҸӘе…іеҝғиҝҷдәӣж“ҚдҪңжңүжІЎжңүз»“жқҹгҖӮиҝҷж—¶пјҢPromise.allSettled()ж–№жі•е°ұеҫҲжңүз”ЁгҖӮеҰӮжһңжІЎжңүиҝҷдёӘж–№жі•пјҢжғіиҰҒзЎ®дҝқжүҖжңүж“ҚдҪңйғҪз»“жқҹпјҢе°ұеҫҲйә»зғҰгҖӮPromise.all()ж–№жі•ж— жі•еҒҡеҲ°иҝҷдёҖзӮ№гҖӮ

    Promise.any()

    return any resolved

    еҸӘиҰҒеҸӮж•°е®һдҫӢжңүдёҖдёӘеҸҳжҲҗfulfilledзҠ¶жҖҒпјҢеҢ…иЈ…е®һдҫӢе°ұдјҡеҸҳжҲҗfulfilledзҠ¶жҖҒпјӣеҰӮжһңжүҖжңүеҸӮж•°е®һдҫӢйғҪеҸҳжҲҗrejectedзҠ¶жҖҒпјҢеҢ…иЈ…е®һдҫӢе°ұдјҡеҸҳжҲҗrejectedзҠ¶жҖҒгҖӮиҜҘж–№жі•зӣ®еүҚжҳҜдёҖдёӘз¬¬дёүйҳ¶ж®өзҡ„жҸҗжЎҲ гҖӮ

    иҝ”еӣһдёҖдёӘж–°зҡ„ Promise е®һдҫӢ

    Promise.resolve()

    йңҖиҰҒе°ҶзҺ°жңүеҜ№иұЎиҪ¬дёә Promise еҜ№иұЎгҖӮPromise.resolve()еңЁжң¬иҪ®вҖңдәӢд»¶еҫӘзҺҜвҖқз»“жқҹж—¶жү§иЎҢ

    Promise.reject()

    Promise.reject(reason)ж–№жі•д№ҹдјҡиҝ”еӣһдёҖдёӘж–°зҡ„ Promise е®һдҫӢпјҢиҜҘе®һдҫӢзҡ„зҠ¶жҖҒдёәrejectedгҖӮ
    жіЁж„ҸпјҢPromise.reject()ж–№жі•зҡ„еҸӮж•°пјҢдјҡеҺҹе°ҒдёҚеҠЁең°дҪңдёәrejectзҡ„зҗҶз”ұпјҢеҸҳжҲҗеҗҺз»ӯж–№жі•зҡ„еҸӮж•°гҖӮиҝҷдёҖзӮ№дёҺPromise.resolveж–№жі•дёҚдёҖиҮҙгҖӮ

    Promise.try()

    е®һйҷ…ејҖеҸ‘дёӯпјҢз»ҸеёёйҒҮеҲ°дёҖз§Қжғ…еҶөпјҡдёҚзҹҘйҒ“жҲ–иҖ…дёҚжғіеҢәеҲҶпјҢеҮҪж•°fжҳҜеҗҢжӯҘеҮҪж•°иҝҳжҳҜејӮжӯҘж“ҚдҪңпјҢдҪҶжҳҜжғіз”Ё Promise жқҘеӨ„зҗҶе®ғгҖӮеӣ дёәиҝҷж ·е°ұеҸҜд»ҘдёҚз®ЎfжҳҜеҗҰеҢ…еҗ«ејӮжӯҘж“ҚдҪңпјҢйғҪз”Ёthenж–№жі•жҢҮе®ҡдёӢдёҖжӯҘжөҒзЁӢпјҢз”Ёcatchж–№жі•еӨ„зҗҶfжҠӣеҮәзҡ„й”ҷиҜҜгҖӮдёҖиҲ¬е°ұдјҡйҮҮз”ЁдёӢйқўзҡ„еҶҷжі•гҖӮ

    дёҠйқўзҡ„еҶҷжі•жңүдёҖдёӘзјәзӮ№пјҢе°ұжҳҜеҰӮжһңfжҳҜеҗҢжӯҘеҮҪж•°пјҢйӮЈд№Ҳе®ғдјҡеңЁжң¬иҪ®дәӢд»¶еҫӘзҺҜзҡ„жң«е°ҫжү§иЎҢгҖӮ

    дёҠйқўд»Јз ҒдёӯпјҢеҮҪж•°fжҳҜеҗҢжӯҘзҡ„пјҢдҪҶжҳҜз”Ё Promise еҢ…иЈ…дәҶд»ҘеҗҺпјҢе°ұеҸҳжҲҗејӮжӯҘжү§иЎҢдәҶгҖӮ

    йӮЈд№ҲжңүжІЎжңүдёҖз§Қж–№жі•пјҢи®©еҗҢжӯҘеҮҪж•°еҗҢжӯҘжү§иЎҢпјҢејӮжӯҘеҮҪж•°ејӮжӯҘжү§иЎҢпјҢе№¶дё”и®©е®ғд»¬е…·жңүз»ҹдёҖзҡ„ API е‘ўпјҹеӣһзӯ”жҳҜеҸҜд»Ҙзҡ„пјҢе№¶дё”иҝҳжңүдёӨз§ҚеҶҷжі•гҖӮ

    з¬¬дёҖз§ҚеҶҷжі•жҳҜз”ЁasyncеҮҪж•°жқҘеҶҷгҖӮ

    дёҠйқўд»Јз ҒдёӯпјҢз¬¬дәҢиЎҢжҳҜдёҖдёӘз«ӢеҚіжү§иЎҢзҡ„еҢҝеҗҚеҮҪж•°пјҢдјҡз«ӢеҚіжү§иЎҢйҮҢйқўзҡ„asyncеҮҪж•°пјҢеӣ жӯӨеҰӮжһңfжҳҜеҗҢжӯҘзҡ„пјҢе°ұдјҡеҫ—еҲ°еҗҢжӯҘзҡ„з»“жһңпјӣеҰӮжһңfжҳҜејӮжӯҘзҡ„пјҢе°ұеҸҜд»Ҙз”ЁthenжҢҮе®ҡдёӢдёҖжӯҘпјҢе°ұеғҸдёӢйқўзҡ„еҶҷжі•гҖӮ

    йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢasync () => f()дјҡеҗғжҺүf()жҠӣеҮәзҡ„й”ҷиҜҜгҖӮжүҖд»ҘпјҢеҰӮжһңжғіжҚ•иҺ·й”ҷиҜҜпјҢиҰҒдҪҝз”Ёpromise.catchж–№жі•гҖӮ

    з¬¬дәҢз§ҚеҶҷжі•жҳҜдҪҝз”Ёnew Promise()гҖӮ

    дёҠйқўд»Јз Ғд№ҹжҳҜдҪҝз”Ёз«ӢеҚіжү§иЎҢзҡ„еҢҝеҗҚеҮҪж•°пјҢжү§иЎҢnew Promise()гҖӮиҝҷз§Қжғ…еҶөдёӢпјҢеҗҢжӯҘеҮҪж•°д№ҹжҳҜеҗҢжӯҘжү§иЎҢзҡ„гҖӮ

    йүҙдәҺиҝҷжҳҜдёҖдёӘеҫҲеёёи§Ғзҡ„йңҖжұӮпјҢжүҖд»ҘзҺ°еңЁжңүдёҖдёӘжҸҗжЎҲпјҢжҸҗдҫӣPromise.tryж–№жі•жӣҝд»ЈдёҠйқўзҡ„еҶҷжі•гҖӮ

    дәӢе®һдёҠпјҢPromise.tryеӯҳеңЁе·Ід№…пјҢPromise еә“BluebirdгҖҒQе’ҢwhenпјҢж—©е°ұжҸҗдҫӣдәҶиҝҷдёӘж–№жі•гҖӮ

    з”ұдәҺPromise.tryдёәжүҖжңүж“ҚдҪңжҸҗдҫӣдәҶз»ҹдёҖзҡ„еӨ„зҗҶжңәеҲ¶пјҢжүҖд»ҘеҰӮжһңжғіз”Ёthenж–№жі•з®ЎзҗҶжөҒзЁӢпјҢжңҖеҘҪйғҪз”ЁPromise.tryеҢ…иЈ…дёҖдёӢгҖӮиҝҷж ·жңүи®ёеӨҡеҘҪеӨ„пјҢе…¶дёӯдёҖзӮ№е°ұжҳҜеҸҜд»ҘжӣҙеҘҪең°з®ЎзҗҶејӮеёёгҖӮ

    function getUsername(userId) {
    return database.users.get({id: userId})
    .then(function(user) {
    return user.name;
    });
    }
    дёҠйқўд»Јз ҒдёӯпјҢdatabase.users.get()иҝ”еӣһдёҖдёӘ Promise еҜ№иұЎпјҢеҰӮжһңжҠӣеҮәејӮжӯҘй”ҷиҜҜпјҢеҸҜд»Ҙз”Ёcatchж–№жі•жҚ•иҺ·пјҢе°ұеғҸдёӢйқўиҝҷж ·еҶҷгҖӮ

    database.users.get({id: userId})
    .then(вҖҰ)
    .catch(вҖҰ)
    дҪҶжҳҜdatabase.users.get()еҸҜиғҪиҝҳдјҡжҠӣеҮәеҗҢжӯҘй”ҷиҜҜпјҲжҜ”еҰӮж•°жҚ®еә“иҝһжҺҘй”ҷиҜҜпјҢе…·дҪ“иҰҒзңӢе®һзҺ°ж–№жі•пјүпјҢиҝҷж—¶дҪ е°ұдёҚеҫ—дёҚз”ЁtryвҖҰcatchеҺ»жҚ•иҺ·гҖӮ

    try {
    database.users.get({id: userId})
    .then(вҖҰ)
    .catch(вҖҰ)
    } catch (e) {
    // вҖҰ
    }
    дёҠйқўиҝҷж ·зҡ„еҶҷжі•е°ұеҫҲз¬ЁжӢҷдәҶпјҢиҝҷж—¶е°ұеҸҜд»Ҙз»ҹдёҖз”Ёpromise.catch()жҚ•иҺ·жүҖжңүеҗҢжӯҘе’ҢејӮжӯҘзҡ„й”ҷиҜҜгҖӮ

    Promise.try(() => database.users.get({id: userId}))
    .then(вҖҰ)
    .catch(вҖҰ)
    дәӢе®һдёҠпјҢPromise.tryе°ұжҳҜжЁЎжӢҹtryд»Јз Ғеқ—пјҢе°ұеғҸpromise.catchжЁЎжӢҹзҡ„жҳҜcatchд»Јз Ғеқ—гҖӮ
